Project Overview

Understanding the objectives, datasets and methodology used in this project.

Project Description

Urban expansion is one of the most significant indicators of territorial transformation. This project evaluates how urban areas have expanded over time using satellite imagery, GIS techniques and spatial analysis.

The study combines remote sensing with geographic information systems to identify new urban zones, compare historical changes and support sustainable planning.

Study Area Casablanca, Morocco
Software QGIS • Python • Google Earth Engine
Satellite Data Sentinel-2 / Landsat
Coordinate System WGS 84 UTM Zone 29N
